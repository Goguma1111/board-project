# 🎀 몽글몽글 게시판 (Mongle Board) ✨

> **오늘의 소소하고 즐거운 일상들을 함께 나누는 공간이에요!** 🌸  
> 파스텔 톤의 귀여운 디자인과 공지사항, 회원가입/로그인 기능을 갖춘 게시판 프로젝트입니다.
> 아직 수정 중 입니다. :)

---

## 🎨 주요 기능 (Features)

- 📢 **공지사항 시스템**: 상단에 포근한 개나리 톤으로 최신 공지글 **최대 5개** 표출!
- 📝 **게시글 작성 및 조회**: 일상글을 작성하고 동글동글한 버블 카드로 게시글 확인
- 🔍 **검색 기능**: 원하는 키워드로 게시글 빠르게 찾기
- 💌 **회원가입 & 로그인**: 유저 인증 및 내 정보 유지 기능

## 실행방법 

 cd backend
### Gradle 빌드 및 실행
./gradlew bootRun


cd frontend
### 패키지 설치 및 서버 실행
npm install
npm run dev
